CSS as flow content test

This tests whether an HTML document providing CSS in a style element as flow content can be used as a style sheet.

If the text is green, and each paragraph is surrounded by a dotted green border, the browser does use the linked HTML flow content as a style sheet - but in practice Firefox, Safari, and Yandex browser ignore the first rule in the linked stylesheet - I have to check if that's a bug or some counterintuitive specified behaviour.
